# Advanced Features

- Circuit Boards: Nestable, editable via GUI. Support caching/pre-simulation for performance.
- Visual UI: Hologram-style or GUI node editor for building circuits (subject to mod API limits).
- Intuitive Wiring: Needle Tool and Arc Relays to make wiring fast and tidy.
- Movement Tools: Mana-Coil Shifter and Stickythread concepts for moving blocks/entities using mana.

Community Sharing:

- Import/export of circuits via custom workbench and server-hosted JSON codes.
- Website sharing (example codes like "kelpy/FullAdder").

Upgrade Path:

- Components can be upgraded to improve speed, range, or reduce mana use.

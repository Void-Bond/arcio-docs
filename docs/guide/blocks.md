# Blocks & Items

This page lists Arcio's primary items and blocks.

- Shard Catalyst: Portable power/battery that emits a baseline signal value of 1. Rechargeable.
- Etherlamp: Light block that turns on when powered.
- Arc Button: Momentary input — passes energy while pressed.
- Arc Switch: Toggle input — passes energy when on.
- Sigil Matrix: Configurable logic block (AND/OR/XOR/NOT/NAND/NOR/Cohesion).
- Arc Relay: For sharing an input to multiple outputs, organizing Manathreads, and extending range.
- Arc Timer: Configurable delay output (timing ranges set in UI).
- Mana Air Purifier: Early-game passive mana generator.
- Fuel Generators: Mid/late-game higher-output generators using fuel.
- Manathreads: Directional wires for connecting inputs/outputs.
- Circuit Boards: Placeable compact circuits with GUI editing; editable with the Inspector Item.
- Inspector Item: Opens/edits Circuit Boards.
- Needle Tool: Intuitive wiring tool for creating Manathread connections.
- Leylight components: Arc Funnel, Chromatic Emitter, Leylight Mirror, Prism, Focal Array, Command Transceiver, Arc Depot (see Leylight page).

Each block has clear input/output faces and configurable behavior where applicable.
